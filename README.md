# nuxt3-net6-starter

Integrates Nuxt and .NET 6 in a VS Code Workspace setup, with SSL.

Swapped UnoCSS for Tailwind as I prefer the limited classes of Tailwind over Windi.

Kept UnoCSS for its amazing icon generation though.
